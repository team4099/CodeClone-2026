# Code-Clone 2026
Team 4099's learning (cloning) code for the 2026 REBUILT FRC offseason

### For new members:
- Make a **fork** of this repository, and use branches the same way that you would use branches on a real robot repository.
  - AKA, have one branch per subsystem and/or feature.
- Whenever you have a branch completed, send the pull request in #sbt_programming for it to be reviewed. It MUST be reviewed by a veteran member for it to get approved.
  - Don't wait for one PR to be approved to begin work on another one!
- If you have any questions at all, no matter how small, send them in #sbt_programming.
